### Requirement: Asset Management

#### Scenario: View asset details
- **WHEN** a user clicks "详情" (Details) button in the asset list
- **THEN** a modal dialog displays complete asset information including basic info, SSH connection info, hardware specs, tags, and relationships
- **AND** the user can view asset details without leaving the asset list page
- **AND** the user can close the modal to return to the list view
- **AND** the asset detail page route remains available for direct URL access
