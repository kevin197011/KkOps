# Change: Add SSH Session Management UI

## Why
当前 SSH 管理页面只有连接管理和密钥管理两个标签页，缺少会话管理界面。虽然后端 API 已经支持会话管理，但前端用户无法查看和管理 SSH 会话。需要添加会话管理界面，使用户能够：
- 查看所有 SSH 会话列表
- 查看会话详情（连接信息、用户、状态、持续时间等）
- 创建新的 SSH 会话
- 关闭活跃的 SSH 会话
- 按用户、连接、状态过滤会话

## What Changes
- **新增 SSH 会话管理标签页**: 在 SSH 管理页面中添加第三个标签页
- **实现会话列表**: 显示所有会话，支持分页和过滤
- **实现会话创建**: 允许用户创建新的 SSH 会话
- **实现会话关闭**: 允许用户关闭活跃的会话
- **显示会话详情**: 展示会话的完整信息

## Impact
- **Affected specs**: 
  - `ssh-management` (修改 - 添加前端会话管理需求)
- **Affected code**: 
  - 前端: 更新 SSH.tsx 页面，添加会话管理标签页和相关功能
  - 前端: SSH Service 已存在，无需修改
