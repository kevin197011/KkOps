# Change: 移除项目列表创建时间列

## Why

项目列表页面（`/projects`）不需要显示"创建时间"列，用户希望移除该列以简化界面。

## What Changes

### 前端变更
- 从 `ProjectList.tsx` 的表格列定义中移除"创建时间"列
- 移除相关的格式化逻辑和 moment 导入（如果不再需要）

## Impact

- **受影响的文件**:
  - `frontend/src/pages/projects/ProjectList.tsx`
- **依赖项**: 如果移除创建时间列后不再需要 moment，可以考虑移除导入（但其他列可能仍在使用）

## 非功能性需求

- **简洁性**: 简化表格界面，只显示必要的信息
- **一致性**: 与其他类似的列表页面保持一致（如果有的话）
