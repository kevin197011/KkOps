# Project Management - Time Display Fix

## MODIFIED Requirements

### Requirement: 项目列表创建时间格式化显示

项目列表页面的"创建时间"列 **SHALL** 显示格式化的日期时间，而非原始 ISO 8601 时间戳。

**修改前**: 创建时间列显示原始 ISO 8601 时间戳格式（如 `2025-01-19T12:00:00Z`）

**修改后**: 创建时间列 **SHALL** 显示格式化的日期时间（如 `2025-01-19 12:00:00`）

#### Scenario: 查看项目列表时显示格式化的创建时间
1. 用户访问 `/projects` 页面
2. 系统显示项目列表表格
3. **期望**: "创建时间"列显示格式化的日期时间字符串 `YYYY-MM-DD HH:mm:ss`
4. **期望**: 如果创建时间为空或无效，显示 `-`

#### Scenario: 时间格式与项目其他页面保持一致
1. 用户在项目列表页面查看创建时间
2. 用户在任务管理页面查看时间字段
3. **期望**: 所有页面的时间格式保持一致（`YYYY-MM-DD HH:mm:ss`）
