# Change: 修复项目列表创建时间显示

## Why

项目列表页面（`/projects`）的"创建时间"列定义了但未格式化显示，导致时间戳以原始 ISO 8601 格式显示（如 `2025-01-19T12:00:00Z`），用户体验不佳。需要格式化为易读的日期时间格式（如 `2025-01-19 12:00:00`）。

## What Changes

### 前端变更
- 修复 `ProjectList.tsx` 中创建时间列的显示格式
- 使用 `moment` 库格式化时间戳为 `YYYY-MM-DD HH:mm:ss` 格式
- 处理空值或无效时间的情况，显示 `-` 或友好提示

## Impact

- **受影响的文件**:
  - `frontend/src/pages/projects/ProjectList.tsx`
- **依赖项**: 项目已安装 `moment` 库，无需新增依赖

## 非功能性需求

- **性能**: 时间格式化应在渲染时进行，对性能影响可忽略
- **一致性**: 时间格式应与项目中其他页面（如 `ScheduledTaskList.tsx`）保持一致
- **兼容性**: 支持后端返回的各种时间格式（ISO 8601、RFC 3339 等）
