# Change: Add Host Group and Tag Management Pages

## Why
当前主机管理功能支持主机分组和标签，但缺少专门的管理界面。管理员需要能够：
- 创建、编辑、删除主机组
- 创建、编辑、删除主机标签
- 查看主机组和标签的详细信息
- 在主机管理页面中方便地为主机分配组和标签

## What Changes
- **新增主机组管理页面**: 提供主机组的完整 CRUD 操作
- **新增主机标签管理页面**: 提供主机标签的完整 CRUD 操作
- **增强主机管理页面**: 在主机列表和编辑表单中支持组和标签的分配
- **后端 API 完善**: 如果缺失，添加主机组和标签的 CRUD API

## Impact
- **Affected specs**: 
  - `host-management` (修改 - 添加分组和标签管理需求)
- **Affected code**: 
  - 后端: 可能需要添加主机组和标签的 CRUD API（如果缺失）
  - 前端: 新增主机组管理页面和主机标签管理页面
  - 前端: 更新主机管理页面，添加组和标签分配功能
