# KkOps Frontend

智能运维管理平台前端应用

## 技术栈

- React 18+
- TypeScript
- Ant Design 5+
- Vite
- React Router v6
- Axios

## 开发环境设置

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 项目结构

```
frontend/
├── src/
│   ├── api/         # API接口定义
│   ├── components/  # 公共组件
│   ├── layouts/     # 布局组件
│   ├── pages/       # 页面组件
│   ├── stores/      # 状态管理
│   ├── hooks/       # 自定义Hooks
│   ├── utils/       # 工具函数
│   └── themes/      # 主题配置
└── public/          # 静态资源
```
